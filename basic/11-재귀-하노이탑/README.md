## 재귀-하노이탑

- 하노이탑 규칙

  - 한 번에 하나의 원반을 움직일 수 있다.
  - 가장 위에 있는 원반만 옮길 수 있다.
  - 아래에 작은 원반이 올 수 없다.

- 하노이탑 구현

  - [Code Link](../dev/hanoi.mjs)

- 위 구현 코드 속 함수가 어떻게 작동하는지 살펴보자.

  ```js
  function hanoi(count, from, to, temp) {
    if (count == 0) return;
    hanoi(count - 1, from, temp, to);
    console.log(`원반 ${count}를 ${from}에서 ${to}로 이동`);
    hanoi(count - 1, temp, to, from);
  }

  // 아래 함수의 작동 과정을 자세히 살펴보자.
  hanoi(3, "A", "C", "B");

  if (1 == 0) return;
  hanoi(1 - 1, "A", "B", "C");
  if (0 == 0) return; // 참이므로 함수 종료하고 이 함수를 호출했던 함수로 실행이 넘어간다.
  console.log(`원반 ${1}를 ${"A"}에서 ${"C"}로 이동`);
  ```
